## 컨셉

1.  메모버튼을 누른다
2.  메모내용과 상세내용을 입력한 뒤 위치를 탭이나 시간알림 탭을 누른다.
3.  위치탭을 눌렀을대는 위치를 입력한다. 예를들면 가게이름이면 다이소, 버거킹
4.  반경을 설정한다. 5.저장을 한다.
5.  실제로 유저는 이동을 한다 7.이동중 반경내에 메모로 설정한 이름 (e.g 다이소, 버거킹) 이 있으면 알림을 보내준다
